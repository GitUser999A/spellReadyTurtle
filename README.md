SpellReady (Turtle WoW) Quick Setup

1) How to install
Put folder SpellReadyTurtle into: World of Warcraft\Interface\AddOns\
Start game and enable addon in AddOns list.

2) How to configure
Type: /sr or /spellready in chat

3) Basic use
Pick your Style:
Style A = centered pulse (icon/text/both)
Style B = icon row that reacts to cooldowns

Check spells you want to track.
Press Test to preview.
Adjust options (size/fade/font etc.).
Use Edit Position (Style A) or drag row (Style B), then Lock Position.

4) In case of issues
If /sr does nothing
Type /reload
Try /sr again
If still broken, enable Lua errors:
/console scriptErrors 1
then reload and check the red error message.
